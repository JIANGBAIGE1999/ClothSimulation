个人开发布料模拟，目前已开发功能
1. 粒子(布料)顶点系统的安装，主要采取了在布料(PLANE)上均匀弥补的添加骨骼
2. 弹簧系统
3. 布料顶点的预测和弹簧系统的模拟
4. 离散型碰撞检测，就是与SphereCollider, BoxCollider和CapsuleCollider的碰撞检测和碰撞相应
5. 布料自相交(可检测出自相交，但碰撞相应出现问题，目前正在修改)
